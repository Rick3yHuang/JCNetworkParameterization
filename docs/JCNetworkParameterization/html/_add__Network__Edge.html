addNetworkEdge -- Add an edge by dividing two existing edges and add a new pair of reticulation edges to a Network data type.

  • Usage:
    addNetworkEdge(N,edgesToDivide,vertexInNewReticulation)
    addNetworkEdge(N,edgesToDivideList,vertexInNewReticulationList)
  • Inputs:
    • N, an instance of the type Network, a Network data type including the sorted edges, leaves, reticulation edges, and level of the network
    • edgesToDivide, a list, a list of two edges to divide when adding a new edge
    • vertexInNewReticulation, an integer, an integer representing the vertex in the new reticulation edge. This is the source vertex for one of the reticulation edges
    • edgesToDivideList, a list, a list of lists of two edges to divide when adding new edges
    • vertexInNewReticulationList, a list, a list of integers representing the vertices in the new reticulation edges
  • Outputs:
    • NNew, an instance of the type Network, a Network data type including the updated sorted edges, leaves, reticulation edges, and level of the network

Description

The following example adds a new reticulation edge to a given Network data type by dividing two existing edges.

i1 : edges = {{1,8},{2,7},{3,6},{4,5},{5,6},{6,7},{7,8},{5,8}};
 
i2 : reticulations = {{{6,7},{7,8}}};
 
i3 : leaves = {1,2,3,4};
 
i4 : exampleNetwork = getNetwork(edges,leaves,reticulations);
 
i5 : exampleNetwork2 = addNetworkEdge(exampleNetwork,{{1,8},{7,2}},7);

i9 : leaves = {1,2,3,4};
