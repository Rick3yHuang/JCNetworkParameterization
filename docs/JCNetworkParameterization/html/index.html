JCNetworkParameterization -- A Macaulay2 package to work with Phylogenetics Identifiability.

Description

JCNetworkParameterization is a package that can find the parameterization of a given network

Menu

  • addNetworkEdge -- Add an edge by dividing two existing edges and add a new pair of reticulation edges to a Network data type.
  • computeParameterization -- Compute the parameterization of a given network under a given model
  • computeDimensionNumerically -- Compute the dimension of a given parameterization numerically
  • Network -- A general datatype for a network
  • Model -- A general datatype for a model
  • getModel -- A constructor method for the Model data type
  • getNucleotideSequence -- An accessor method for the nucleotide sequence of a Model data type
  • getTransformTable -- An accessor method for the transform table of a Model data type
  • getNetwork -- A constructor method for the Network data type
  • getReticulationEdges -- An accessor method for the reticulation edges of a Network data type
  • getEdges -- An accessor method for the edges of a Network data type
