getTransformTable -- An accessor method for the transform table of a Model data type

  • Usage:
    getTransformTable(Model)
  • Inputs:
    • M, an instance of the type Model, a Model data type including the nucleotide sequence and transform table
  • Outputs:
    • transformTable, a hash table, a hash table indicating the representatives and converters for the model

Description

This method retrieves the hash table from a given Model data type.

i1 : y3L = hashTable{A => 0, C => 1, G => 2, T => 3};
 
i2 : L3L = {(A,A,A),(A,C,C),(C,A,C),(C,C,A),(C,G,T)};
 
i3 : M3L = getModel(L3L,y3L);
 
i4 : getTransformTable M3L

o4 = HashTable{A => 0}
               C => 1
               G => 2
               T => 3

o4 : HashTable
