getReticulationEdges -- An accessor method for the reticulation edges of a Network data type

  • Usage:
    getReticulationEdges(Network)
  • Inputs:
    • N, an instance of the type Network, a Network data type including the sorted edges, leaves, reticulation edges, and level
  • Outputs:
    • reticulationEdges, a list, a list of reticulation edges in the network, where each reticulation edge is represented as a list of two vertices

Description

This method retrieves the reticulation edges as a vertical list from a given Network data type.

i1 : leaves3LL2 = {1,2,3};
 
i2 : EPList3LL2 = {{2,7},{8,3},{4,5},{4,1},{4,6},{6,7},{5,8},{7,8},{5,6}};
 
i3 : reticulationPairList3LL2 = {{{4,6},{5,6}},{{6,7},{7,8}}};
 
i4 : N3LL2 = getNetwork(EPList3LL2,leaves3LL2,reticulationPairList3LL2);
 
i5 : getReticulationEdges N3LL2

o5 = {{{4, 6}, {5, 6}}}
     {{{6, 7}, {7, 8}}}

o5 : VerticalList
