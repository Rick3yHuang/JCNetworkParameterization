computeParameterization -- Compute the parameterization of a given network under a given model

  • Usage:
    computeParameterization(N,M)
  • Inputs:
    • N, an instance of the type Network, a network indicating the edges, leaves, reticulation edges, and level of the network.
    • M, an instance of the type Model, a model including the information about the representatives and converters
  • Optional inputs:
    • includeQs => a Boolean value, default value true, a boolean value indicating whether the Fourier coordinates should be included in the parameterization
  • Outputs:
    • parameterization, a list, a list of polynomials from the parameterization

Description

Let N3LL2 be a 3-leaf level-2 network and M3L be a Jukes-Canter phylogenetic model. This function computes the parameterization of N3LL2 under the model M3L. The output is a list of polynomials representing the parameterization.
