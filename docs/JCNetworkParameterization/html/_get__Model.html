getModel -- A constructor method for the Model data type

  • Usage:
    getModel(necleotideSequence,transformTable)
  • Inputs:
    • necleotideSequence, a list, a list of nucleotides used in the model
    • transformTable, a hash table, a hash table indicating the representatives and converters for the model
  • Outputs:
    • M, an instance of the type Model, a Model data type including the nucleotide sequence and transform table

Description

The following example constructs a Model data type for the Jukes-Cantor model.

i1 : y3L = hashTable{A => 0, C => 1, G => 2, T => 3};
 
i2 : L3L = {(A,A,A),(A,C,C),(C,A,C),(C,C,A),(C,G,T)};
 
i3 : M3L = getModel(L3L,y3L)

o3 = Model{...2...}

o3 : Model

Ways to use getModel:

  • getModel(List,HashTable)

For the programmer

The object getModel is a method function.
